🛠️ Troubleshooting

If mobile viewer doesn't load:

  1. Make sure your server is running on localhost:8002
  2. Try the direct URLs above
  3. Check browser console for any errors
  4. Try opening in mobile device or device emulator

Best testing approach:

  • Open browser dev tools (F12)
  • Toggle device toolbar (mobile view)
  • Select iPhone or Android device
  • Test touch gestures and mobile features
