üìã Case Study: 1955 Zenith Space Command Remote Control TV Restoration

Challenge: Rare 1955 Zenith Space Command television with ultrasonic remote control system. Non-functional with multiple component failures and severe cabinet damage from storage.

Restoration Challenges:

  • Unique ultrasonic remote control system requiring specialized parts
  • CRT picture tube showing age-related brightness loss
  • Multiple failed electrolytic capacitors affecting all circuits
  • Damaged mahogany cabinet requiring period-correct refinishing
  • Rare "Space Command" remote with non-functional ultrasonics
  • Historical significance requiring museum-quality restoration

Our Comprehensive Restoration:

  • Sourced original Zenith ultrasonic receiver modules from NOS dealer
  • CRT rejuvenation using specialized tube restoration equipment
  • Complete recap with period-appropriate Sprague capacitors
  • Cabinet restoration using authentic tung oil and traditional techniques
  • Remote control restoration with ultrasonic tuning fork replacement
  • Created comprehensive documentation for collector's records

Result: Fully functional 1955 Zenith with working ultrasonic remote, museum-quality cabinet restoration, all original functionality restored, appraised value increased from $500 to $4,500.
